§0 — Reduction Principle

The system is defined by a single test: what structure can be removed while the same canonical unfold, replay, and projection still results? A structure is foundational if and only if removing it changes canonical output. Everything else is either derived from the foundation or projected from it.

The specification is organized as five laws, each decomposed into one or more executable procedures (A1–A11). Laws are normative. Procedures are the computable form of each law. Invariants I1–I8 are stated separately. Witnesses (73, B, W, the Fano line choice) are non-normative — derived from the laws and frozen in artifacts.

No claim below is admitted by argument. Each is verifiable by running the frozen test suite against the frozen artifacts.

§1 — Law I · Kernel Transition (A1–A2)

-- state space 𝕊ₙ = { x ∈ ℤ | 0 ≤ x < 2ⁿ } maskₙ = 2ⁿ − 1 -- rotation operators on 𝕊ₙ rotl(x, k, n) = ((x << k) | (x >> (n−k))) & maskₙ rotr(x, k, n) = ((x >> k) | (x << (n−k))) & maskₙ -- constant: 0x1D repeated for each byte of width n Cₙ = ⊕ᵢ₌₀ⁿ⁻¹ 0x1D << (8i) -- n=16: 0x1D1D n=32: 0x1D1D1D1D -- n=64: 0x1D1D1D1D1D1D1D1D
A1 delta 𝕊ₙ × Cₙ × ℕ⁺ → 𝕊ₙ 
delta(x, C, n):
  return (rotl(x,1,n)  XOR
          rotl(x,3,n)  XOR
          rotr(x,2,n)  XOR
          C           )  AND  maskₙ
A2 replay 𝕊ₙ × ℕ × Cₙ × ℕ⁺ → 𝕊ₙ* 
replay(x₀, steps, C, n):
  x      := x₀ AND maskₙ
  result := []
  repeat steps times:
    result.append(x)
    x := delta(x, C, n)
  return result

§2 — Law II · Mixed-Radix Coordinate (A3–A6)

A3 mixed_encode ℤ≥₀ × R* → ℤ≥₀* 
mixed_encode(v, R):            -- R = [r₀, r₁, …, rₖ₋₁], each rᵢ ≥ 2
  coords := []
  for r in R:
    coords.append(v mod r)
    v := floor(v / r)
  coords.append(v)             -- most-significant remainder
  return coords                -- |coords| = |R| + 1
A4 mixed_decode ℤ≥₀* × R* → ℤ≥₀ 
mixed_decode(coords, R):
  v := coords[|R|]             -- start from most-significant
  for i from |R|−1 down to 0:
    v := coords[i] + R[i] × v
  return v
-- Theorem 4 (Reversibility) ∀ v ≥ 0, ∀ R with each rᵢ ≥ 2: mixed_decode( mixed_encode(v, R), R ) = v

Canonical instances

-- orbit / offset (crystal timing axis) R = [W] = [36] encode(pos, [36]) → [offset, orbit] where orbit = pos div 36 offset = pos mod 36 -- channel / lane (control-plane address) R = [16] encode(addr, [16]) → [lane, channel] -- Unicode codepoint (plane / offset decomposition) R = [65536] encode(cp, [65536]) → [offset, plane] where plane = cp div 65536 ∈ [0..16] offset = cp mod 65536 ∈ [0..65535] decode([offset, plane], [65536]) = plane × 65536 + offset = cp

These three instances are the same algorithm with different radices. Unicode plane decomposition is not a special case; it is an case.

§3 — Law III · Basis Projection (A5–A6)

A5 project_value ℤ≥₀ × plane × basis_spec → repr 
project_value(v, p, b):
  b.kind = "2"         →  ("2",          v)
  b.kind = "8"         →  ("8",          v)
  b.kind = "10"        →  ("10",         v)
  b.kind = "16"        →  ("16",         v)
  b.kind = "36"        →  ("36",         v)
  b.kind = "codepoint" →  ("codepoint", [v mod 65536,  v div 65536])
  b.kind = "mixed"     →  ("mixed",      mixed_encode(v, b.radices))
A6 interpret_value repr × plane × basis_spec → ℤ≥₀ 
interpret_value(repr, p, b):
  ("2",          v)      →  v
  ("8",          v)      →  v
  ("10",         v)      →  v
  ("16",         v)      →  v
  ("36",         v)      →  v
  ("codepoint", [o, pl]) →  pl × 65536 + o
  ("mixed",      coords) →  mixed_decode(coords, b.radices)
-- Theorem 5 (Projection Law) ∀ v ≥ 0, ∀ valid (p, b): interpret_value( project_value(v, p, b), p, b ) = v

§4 — Law IV · Incidence Coordination (A7–A8)

The seven structural token types form the points of the Fano plane PG(2,2): 7 points, 7 lines, 3 points per line, 3 lines per point. The seven lines define a deterministic 7-periodic schedule over ticks. fano_triplet is the canonical incidence scheduler for this version. The schedule is replaceable in principle — the seven-type partition (from Law III) is the irreducible structure; the specific 7-cycle covering is one valid ordering of it.

§5 — Law IV · Structural Plane Projection (A9–A10)

Each entity carries a canonical byte stream. That stream is projected through four structural planes. The collapse condition is a structural property of the stream's content, not an assigned label.

The boundary byte values below (0x80, 0xA0, 0xC0, 0xE0) are the reference encoding from docs/CONTROL_PLANE_SPEC.md — the canonical control-plane byte assignments for FS, GS, RS, and US channel boundaries at lane 0, TYPE=0. They are not an implementation choice; they follow from the FLAG-bit partition and the CH field encoding specified in that document.

A9 projection_vector bytes × frame → bytes⁴ 
-- PLANE_OPEN: FS→0x80, GS→0xA0, RS→0xC0, US→0xE0

project_entity(stream, plane):
  boundary := PLANE_OPEN[plane]
  inside   := false
  content  := []
  for b in stream:
    is_ctrl  := (b AND 0x80) ≠ 0
    is_bdry  := is_ctrl AND (b AND 0xE0) = (boundary AND 0xE0) AND (b AND 0x01) = 0
    if is_bdry:   inside := NOT inside
    elif NOT is_ctrl AND inside:  content.append(b)
  return bytes(content)

projection_vector(stream, frame):
  return [
    project_entity(stream, FS),
    project_entity(stream, GS),
    project_entity(stream, RS),
    project_entity(stream, US),
  ]
A10 is_collapsed · continuation_surface · step_projection bytes⁴ → bool / bytes* 
is_collapsed(V):
  return  V[0] = V[1] = V[2] = V[3]

continuation_surface(V):
  seen := []
  for x in V:
    if x ∉ seen:  seen.append(x)
  return seen                        -- deduplicated, order-preserving

step_projection(stream, tick, basis_spec):
  f := frame_at_tick(tick, basis_spec, RS)
  V := projection_vector(stream, f)
  if is_collapsed(V):
    return ("collapsed",  V[0])
  else:
    return ("divergent",  continuation_surface(V))
-- example: pure payload (no structural markers) stream = [0x42, 0x43, 0x44] -- FLAG=0 bytes only V = [b'BCD', b'BCD', b'BCD', b'BCD'] is_collapsed(V) = true -- example: structured stream (GS and US markers) stream = [0xA0, 0x42, 0x43, 0xA0, 0xE0, 0x44, 0xE0] V = [b'', b'BC', b'', b'D'] is_collapsed(V) = false continuation_surface(V) = [b'', b'BC', b'D'] -- |surface| = 3

§6 — Law V · Carrier Verification (A11)

A11 verify_package · apply_package pkg → bool 
pkg := {
  type:             "artifact_package",
  version:          1,
  artifact_kind:    string,
  payload_bytes:    bytes,
  fingerprint_algo: "sha3_256",
  fingerprint:      "sha3_256:" || hex(sha3_256(payload_bytes)),
}

verify_package(pkg):
  computed := "sha3_256:" || hex( sha3_256(pkg.payload_bytes) )
  return computed = pkg.fingerprint

apply_package(pkg):
  if NOT verify_package(pkg):  reject
  parse payload
  validate kind and schema
  apply
